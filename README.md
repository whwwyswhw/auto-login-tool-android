# PK10 彩票投注助手 (Android 版)

基于 Flet 框架的 PK10 彩票投注助手移动端版本。

## 功能特性

- 🔐 自动登录
- 🎱 开奖号码实时显示（图片球）
- 📝 投注方案管理
- 🌙 深色主题
- 📊 操作日志

## 安装

### 方式一：下载 APK（推荐）

前往 [Releases](https://github.com/whwwyswhw/auto-login-tool-android/releases) 页面下载最新 APK。

### 方式二：从源码运行

```bash
pip install flet requests pillow
flet run main.py
```

## 构建 APK

```bash
pip install flet[build]
flet build apk
```

## 技术栈

- Python 3.8+
- Flet (Flutter)
- requests
- PIL (Pillow)
